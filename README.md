# 🍳 레시피튜브 (Recipe Tube)

유튜브 인기 요리 영상을 검색하면 **성과 좋은 상위 영상**이 뜨고, 각 영상의
**재료·레시피·조리법을 자동 요약**해 보여주는 웹앱입니다. 마음에 드는 영상은
**별표(⭐)로 저장**하고, 다음에 **같은/비슷한 메뉴를 검색하면 저장한 영상이 맨 위**에 나옵니다.
PC·모바일 브라우저 모두 지원하고, 휴대폰 홈화면에 앱처럼 설치(PWA)할 수 있어요.

---

## ✨ 주요 기능

- 🔎 **유튜브식 검색** — 검색어를 넣으면 조회수·좋아요·최신성을 종합한 *성과순*으로 상위 영상 표시 (조회수순/최신순/관련성순 전환 가능)
- 🧾 **재료·조리법 요약** — 영상 설명글과 자막을 모아 재료 목록 / 조리 순서 / 팁으로 정리 (Gemini 무료 AI, 키 없으면 설명글 기반 자동 추출)
- ⭐ **저장(별표)** — 좋은 영상을 저장. 저장 시 메뉴 키워드를 함께 기억
- 🔝 **유사 검색 우선노출** — 저장해 둔 메뉴와 비슷한 검색을 하면 저장 영상이 결과 맨 위 "⭐ 저장함" 영역에 뜸
- 🛒 **장보기 목록** — 레시피의 재료를 한 번에 담고, 인분(배수) 조절 시 분량 자동 환산, 복사/체크 관리
- 🕘 **검색 기록**, 🌙 **다크 모드**, 📱 **모바일 설치(PWA)·오프라인 캐시**

## 🚀 배포 방법 (가장 쉬운 순서)

준비물: 구글 계정 하나면 됩니다. (Google Cloud + Vercel 무료)

### 1단계 — 유튜브 API 키 발급 (필수, 무료)
1. https://console.cloud.google.com 접속 → 상단에서 **새 프로젝트** 생성
2. 검색창에 **"YouTube Data API v3"** → 열고 **[사용]** 클릭
3. 왼쪽 **API 및 서비스 → 사용자 인증 정보 → [사용자 인증 정보 만들기] → API 키**
4. 생성된 키를 복사 (이게 `YOUTUBE_API_KEY`)
   - 무료 할당량: 하루 10,000 유닛(검색 약 100회). 개인·소규모 사용엔 충분합니다.

### 2단계 — Gemini AI 키 발급 (선택, 무료 · 요약 품질↑)
1. https://aistudio.google.com/apikey 접속 → **[Create API key]**
2. 생성된 키 복사 (이게 `GEMINI_API_KEY`)
   - 무료 티어: `gemini-2.5-flash` 기준 분당 요청·하루 요청 제한이 있지만 개인 사용엔 충분
   - **이 키를 넣지 않아도** 영상 설명글에서 재료/순서를 뽑아 동작합니다(품질만 조금 낮아짐)

### 3단계 — Vercel에 올리기
가장 쉬운 방법은 GitHub 연동입니다.

**A. GitHub + Vercel (권장)**
1. 이 `recipe-tube` 폴더를 GitHub 저장소로 올립니다.
2. https://vercel.com → GitHub로 로그인 → **[Add New… → Project]** → 방금 올린 저장소 **Import**
3. 배포 설정은 그대로 두고, **Environment Variables**에 아래를 추가:
   - `YOUTUBE_API_KEY` = (1단계 키)
   - `GEMINI_API_KEY` = (2단계 키, 선택)
   - `GEMINI_MODEL` = `gemini-2.5-flash` (선택)
4. **[Deploy]** → 잠시 뒤 `https://내프로젝트.vercel.app` 주소가 생깁니다. 끝!

**B. 컴퓨터에서 CLI로 (개발자용)**
```bash
npm i -g vercel          # 최초 1회
cd recipe-tube
vercel                   # 안내에 따라 배포
vercel env add YOUTUBE_API_KEY   # 키 등록 (GEMINI_API_KEY도 동일)
vercel --prod            # 실서비스 배포
```

### 4단계 — 로컬에서 먼저 테스트 (선택)
```bash
cd recipe-tube
npm install                  # 외부 의존성 없음(실행 안 해도 됨)
cp .env.example .env         # .env 파일에 실제 키 입력
vercel dev                   # http://localhost:3000
```

## 🗂 폴더 구조
```
recipe-tube/
├── api/
│   ├── search.js       # 유튜브 검색 프록시(성과순 정렬). API 키는 서버에만 보관
│   └── summarize.js    # 설명+자막 → 재료/조리법 요약 (Gemini 무료, 폴백 내장)
├── public/
│   ├── index.html      # 화면 구조
│   ├── styles.css      # 반응형 + 다크모드
│   ├── app.js          # 검색·저장·유사노출·장보기·모달 로직
│   ├── manifest.webmanifest / sw.js / icon.svg   # PWA(설치·오프라인)
├── .env.example        # 환경변수 예시
├── vercel.json         # 함수/캐시 설정
└── package.json
```

## 💰 비용
- 유튜브 API·Gemini·Vercel 모두 **무료 티어**로 운영 가능. 별도 서버·DB 불필요(저장은 브라우저에 보관).
- 사용량이 커지면 유튜브 할당량 증설 또는 Gemini 유료 전환을 고려하세요.

## ⚠️ 참고
- 요약은 외부 라이브러리 없이 유튜브 자막을 직접 읽어옵니다(best-effort). 자막이 없거나 설명이 부실한 영상은 요약이 비어 있을 수 있어요(원본 영상 확인 안내가 표시됩니다).
- 저장 목록·장보기·검색기록은 **그 브라우저에만** 저장됩니다. 기기 간 동기화가 필요하면 로그인/DB(예: Supabase) 연동으로 확장할 수 있어요. (원하시면 다음 단계로 붙여 드릴게요.)
