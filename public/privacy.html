개인정보처리방침

최종 업데이트: 2026-07-05

레시피튜브(이하 "서비스")는 이용자의 개인정보를 소중히 다룹니다. 본 서비스는 별도의 회원가입이나 로그인이 없으며, 이용자를 식별할 수 있는 개인정보를 서버에 수집·저장하지 않습니다.

1. 수집하는 정보

  • 본 서비스는 이름·이메일·전화번호 등 개인정보를 수집하지 않습니다.
  • 저장한 영상, 장보기 목록, 검색 기록, 화면 설정 등은 이용자 기기(브라우저)에만 저장되며(로컬 저장소), 서버로 전송되지 않습니다.

2. 외부 서비스

서비스 제공을 위해 아래 외부 서비스를 이용합니다. 각 서비스의 데이터 처리는 해당 사업자의 정책을 따릅니다.

  • YouTube Data API — 요리 영상 검색·정보 표시 (Google 개인정보처리방침 적용)
  • Google Gemini API — 영상 기반 재료·조리법 요약 생성
  • 쿠팡 파트너스 — 장보기의 재료 구매 링크·배너. 링크를 통한 구매 시 서비스가 일정액의 수수료를 받을 수 있습니다.

3. 요약 정보의 캐시

영상 요약 결과는 응답 속도 향상을 위해 서버에 일정 기간 캐시될 수 있습니다. 이 캐시에는 이용자를 식별하는 정보가 포함되지 않습니다.

4. 광고 및 제휴

본 서비스는 쿠팡 파트너스 활동의 일환으로 제휴 링크·배너를 포함하며, 이를 통해 수수료를 제공받을 수 있습니다. 쿠팡 배너는 쿠팡의 정책에 따라 이용자 관심 기반으로 표시될 수 있습니다.

5. 데이터 삭제

기기에 저장된 데이터는 브라우저의 사이트 데이터 삭제(또는 앱 데이터 삭제)로 언제든 완전히 지울 수 있습니다.

6. 아동 보호

본 서비스는 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다.
